# Turing Machine

This program contains an algorithm for a one/two-lane Turing machine to be excecuted and displayed in the terminal.

The program is given the name of a text document that contains the initial conditions and a set of rules for the Turing machine inside the document. It then executes the Turing machine.

The strip can expand to the left/right . If the code detects an infinite strip it halts the program.

By pressing P, you can pause the program/ B to stop the program from excecuting.
